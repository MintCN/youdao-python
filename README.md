# youdao-python
youdao shell implemented in python

## Usage

```sh
$ git clone https://github.com/mudongliang/youdao-python
$ cd youdao-python
$ sudo cp youdao.py /usr/bin/youda

$ which youdao 
/usr/bin/youdao

$ youdao
No word to translate

$ youdao hello
hello
n. 表示问候， 惊奇或唤起注意时的用语
int. 喂；哈罗
n. (Hello)人名；(法)埃洛
```
